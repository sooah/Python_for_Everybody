### How to use SQLite
1. start sqlite
Enter "sqlite3 *.sqlite" ( *: name of file )

2. End sqlite
command + D

3. show tables
~~~sql
.table
~~~

4. show table data
~~~sql
SELECT * FROM tablename;
~~~

5. show data by mode line style
~~~sql
.mode line
~~~

6. show header 
~~~sql
.head on
~~~

7. match column
~~~sql
.mode column
~~~
